1. 설치 File :  setup.exe, MIS.exe
2. 설치환경 :  Windows OS
3. 설치순서
   1) setup.exe file 실행
      a. 32bit windows : C:\Program Files\MIS\ 에 install됨.
      b. 64bit windows : C:\Program Files (x86)\MIS\ 에 install됨.
      바탕화면에 바로가기 메뉴 자동 생성됨
   2) MIS.exe 아래 경로에 덮어쓰기
      a. 32bit windows : C:\Program Files\MIS\ 에 덮어쓰기
      b. 64bit windows : C:\Program Files (x86)\MIS\ 에 덮어쓰기
   3) Auto update
      변경사항은 프로그램실행시 자동 업데이트 되지만, 64bit OS환경에서는 자동업데이트가 안되므로
      수동으로 업데이트 진행하여야 함
4. 프로그램 실행
   바탕화면의 바로가기 메뉴 또는 설치된 경로의 MIS.exe 실행
   최초실행시 DB IP 218.232.105.132 입력
